The Problem

Your Input Is Lying To You

Stick drift sends a constant, low-level signal to your game even when your thumbs are completely still. It's the potentiometers wearing down — every controller does it eventually.

The industry answer is "buy a new one." DriftCore's answer is software. We intercept the raw XInput data, apply a precision dead-zone filter via ViGEmBus, and feed clean virtual input back to your game — transparently, instantly.

Setup Guide

Up & Running In Minutes

01

Install ViGEmBus Driver

DriftCore requires the ViGEmBus virtual controller driver to create a clean output device. Download it from the official GitHub release and run the installer — takes under 60 seconds and only needs to be done once.

ViGEmBus_Setup.exe → Run as Administrator → Restart recommended

A reboot after install ensures the driver loads correctly before you launch DriftCore.

02

Connect Your Xbox Controller

Plug in via USB or connect wirelessly. DriftCore works with any Xbox controller recognised by Windows using the XInput protocol — Xbox One, Xbox Series, or compatible third-party pads.

Settings → Bluetooth & Devices → confirm controller is detected

If Windows doesn't see your controller, verify it works in a game first before launching DriftCore.

03

Launch DriftCore

Run DriftCore.exe — no installation required, it's a standalone executable. Right-click and select Run as Administrator on first launch to allow XInput hooking.

DriftCore.exe → Right-click → Run as Administrator

XInput hooks can trigger antivirus false positives. Whitelist DriftCore.exe in Windows Defender if prompted.

04

Select Your Controller

DriftCore lists all connected XInput devices. Select the drifting controller from the dropdown. The live stick visualiser immediately shows raw input — including the drift signal — in real time.

Controller dropdown → Select device → Live view activates
05

Dial In Your Dead Zone

Use the per-axis sliders to set a dead zone threshold for each stick axis (LX, LY, RX, RY). Start at around 8% and increase until the drift signal disappears from the visualiser. Set it just above the noise floor — too high and you'll lose stick sensitivity near centre.

Recommended starting range: 8–12% on drifting axes

Watch the corrected output in the visualiser. When it holds flat at rest with no thumb on the stick, you're dialled in.

06

Enable Output & Play

Hit Enable to activate the virtual controller. Launch your game — it sees the clean ViGEmBus virtual device instead of your drifting physical controller. No further config needed in-game.

Enable → Launch game → Select controller in game input settings if prompted

Some games require you to manually select the controller. Look for "Xbox Controller 2" or a ViGEm device in the game's input device list.

07

Save Your Profile

Once dialled in, save your settings as a named profile. Enable the Start with Windows option in Settings and DriftCore will auto-load your profile whenever your controller is plugged in — completely hands-free going forward.

File → Save Profile → Settings → Enable "Start with Windows"
